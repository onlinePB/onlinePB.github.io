A bit about myself

Hello! I'm Ryan, a Computing Science student in the UK.
This is my website! I made it to act as a sort of place to collect my various projects. I hope to get a career in software or game development someday, as those areas have really captured my interest. In the meantime, I gotta focus on uni work!

Python I have been doing Python since my third year in secondary school, so naturally I'm quite familiar with it. One of my favourite Python programs I've written was a game made with PyGame I developed for a coursework assignment.
JavaScript JavaScript has been something I've used here-and-there for a while, but once I got into my first year of Uni I really got a grasp on how to use it.
Node.js I've mostly used Node.js for developing Discord bots with libraries like eris or Discord.js, its proven to be very useful to know.
Java I didn't start learning Java until the first year of Uni, and it's been really fascinating to learn! I taught myself how to make graphical games like Pong in Java, and it's really helped my study of the language.

Currently, I hope to learn C# or C++ in future, as they seem like very powerful languages.

Checkout my GitHub profile! This is where I keep my projects, including this site.
